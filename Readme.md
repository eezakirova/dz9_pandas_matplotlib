## Описание
В рамках работы выполнен анализ событий информационной безопасности на основе данных из JSON-файла `events.json`. Анализ проведён с использованием языка Python и библиотек для обработки и визуализации данных.

## Используемые библиотеки
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## Этапы выполнения

**Этап 1. Подготовка данных**  
Загружен файл `events.json`, содержащий события информационной безопасности с полем `signature`.

**Этап 2. Анализ данных**  
Данные из JSON-файла загружены в датафрейм Pandas. Выполнен анализ распределения событий по типам на основе поля `signature`.

**Этап 3. Визуализация данных**  
Построен график распределения типов событий информационной безопасности с использованием Seaborn и Matplotlib.
